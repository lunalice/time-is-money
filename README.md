# TimeIsMoney

## TL;DR
- 昔作成した簡単な時給計算シミュレーターをwebアプリ移植
- データ入力だけで自分のコストパフォーマンス計測出来る！
- DB連携して入力してきた方々の平均値をビビっと出すくらいまでやるかな？

- vue-cli
- python3
- pipenv
